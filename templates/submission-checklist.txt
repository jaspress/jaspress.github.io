SCP异常科学学报 (JSAS) — 投稿检查清单
=====================================================

提交稿件前，请逐项确认以下内容（请在对应项前标注 [✓]）：

【稿件内容】
[ ] 稿件主题在本刊研究范围内
[ ] 研究具有原创性，未在其他期刊发表或同时投稿
[ ] 稿件字数符合对应文章类型的要求

【格式要求】
[ ] 中文摘要不超过400字
[ ] 英文摘要（Abstract）不超过250词
[ ] 关键词3–8个（中英文对照）
[ ] 参考文献采用GB/T 7714–2015格式
[ ] 图片分辨率≥300 DPI，采用TIFF或EPS格式
[ ] 文件为.docx或.tex格式

【双盲评审要求】
[ ] 正文及参考文献中已删除所有可识别作者身份的信息
[ ] 文件属性中已移除作者姓名等个人信息
[ ] 附件（数据文件、代码等）中已做同等匿名处理

【伦理与安全要求】
[ ] 涉及D级人员实验的研究已获伦理委员会审批，并已上传审批文件
[ ] 涉及Keter及以上级异常的研究已填写O5委员会审批编号
[ ] 稿件中不包含可能导致认知危害的内容或图像
[ ] 涉密数据已标注保密级别并附豁免申请（如需）

【声明事项】
[ ] 已填写利益冲突声明
[ ] 所有列名作者均知情并同意投稿
[ ] 如涉及已发表会议摘要，已在稿件中说明

确认完成以上检查后，请通过在线表单提交稿件。
如有疑问，请联系编辑部：editorial@jsas.scp.org
